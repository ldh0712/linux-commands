## 1. top
- 실시간 시스템, 프로세스 정보 모니터링 명령어

### 주요 옵션
| 옵션 | 설명 |
|------|------|
| -u user  | 특정 사용자 프로세스만 표시 |
| -p PID  | 특정 PID만 표시 |
| -n 횟수  | 갱신 횟수 지정 |
| -d 초  | 갱신 주기 설정 |
### 예시 출력
```bash
top
top -u root
top -n 10
```

## 2. ps
- 현재 시스템에서 실행 중인 프로세스를 스냅샷 형태로 출력하는 명령어

### 주요 옵션
| 옵션 | 설명 |
|------|------|
| -e  | 모든 사용자 프로세스 전체 표시 |
| -f  | 표준 전체 포맷으로 프로세스 표시 |
| aux  | BSD 스타일 상세 정보 |
| -u user  | 특정 사용자 프로세스 |

### 사용 예시
```bash
ps -ef
ps aux
ps -u ubuntu
```

## 3. jobs
- 현재 셀에서 실행 중이거나 중지된 백그라운드 작업을 보여준다.
- 시스템 전체가 아닌 현재 터미널만 해당

### 주요 옵션
| 옵션 | 설명 |
|------|------|
| -l  | PID포함 출력 |
| -p  | PID만 출력 |
| -r  | 실행 중인 작업 표시 |
| -s  | 정지된 작업 표시 |

### 사용 예시
```bash
sleep 30 &
jobs
jobs -l
```

---

## 4. kill
- 프로세스를 종료하는 명령어

### 주요 옵션
| 옵션 | 설명 |
|------|------|
| -9   | 강제 종료(SIGKILL) |
| -15  | 정상 종료(SIGTERM, 기본값) |
| -1 | 시그널 목록 출력 |

### 사용 예시
```bash
kill 1234
kill -9 1234
kill -15 1234
kill -l
```
